Detailed Description

Represents implicit matching using if /regex/

if /.*&zwj;/
puts 'true'
else
puts 'false'
end

Prints "false".

Under the hood this construction matches regex against $_, so the following works:

$_ = 'match_me'
if /match_me/
puts 'true'
else
puts 'false'
end

this code prints "true".
